<h4>环境要求：</h4>
<p><mark>Windows8.1/10/11-64位系统（推荐）</mark> 或 Debian8/9/10、Centos7.x/8.x、Mac os系统</p>
<p><mark>Apache2.3+或更高（推荐）</mark> 或 Nginx/Lighttpd</p>
<p><mark>PHP7.4（推荐）</mark> 或 PHP高于7.4的版本</p>
<p><mark>MySql 8.0及以上（推荐）</mark> 或 Mariadb10.5及以上</p>
<h4>注意：</h4>
<ol>
<li>建议部署到windows平台，linux等其他平台部署后可能有未知问题。</li>
<li>参阅下文<strong>Windows自动安装</strong>指南，否则必须自行部署php+mysql环境。</li>
</ol>
<h4>软件特点：</h4>
<ul>
<li>本软件为免费网络共享软件，适用于中小企业会计核算。</li>
<li>软件提供了标准的财务会计序时账簿、总账、明细账、辅助项目、存货核算等功能。</li>
<li>软件框架提供了对数据库的管理，用户角色权限的设置，数据的批量导出等功能。</li>
<li>PHP版采用了javascript + ajax，jquery，mysql存储过程及函数，用户体验极佳。</li>
</ul>
<h4>优势：</h4>
<ul>
<li>安全、高效、自由</li>
<li>开源、免费，软件运作机制公开透明，有强大的开源社区技术支撑。</li>
<li>B/S架构，轻松实现远程多端多用户登录操作。</li>
<li>代码条理清晰、模块化强大。用户可根据自身需求二次开发。</li>
</ul>
<h3>软件架构</h3>
<ul>
<li>软件采用B/S架构，既可以本地运行，也可以多端协作。不但可以企业内网私有化部署，也可以公开部署到云服务器（不推荐）。</li>
<li>路由代码结构模块化，方便二次开发。</li>
<li>前后端合一，避免重复请求，资源负载低。</li>
<li>本软件不需要配置伪静态。</li>
</ul>
<h3>安装教程</h3>
<h4>Windows自动安装：</h4>
<h5><a href="http://www.accsoft2008.com/" title="点这里一键安装Windows罗布会计软件+PHP+Mysql集成环境">去官网下载一键安装包——罗布会计软件+集成环境</a></h5>
<br/>
<hr>
<br/>
<h4>Windows手动安装：</h4>
<ol>
<li>安装好php+mysql环境，并且将项目git clone到站点根目录，<strong>环境要求：需要Windows8/10/11 x64以上、PHP7.4+、Mysql8+环境</strong>。</li>
<li>修改数据库信息，项目路径<code>\config.inc.php</code>的<code>[pwb]</code>字段是数据库密码配置，默认密码为<code>12345678</code>。</li>
<li>然后浏览器访问<code>http://localhost/index.html</code>链接即可。需用域名访问则在web服务器配置上域名并新增域名解析。</li>
<li>~~可将phpMyAdmin部署到<code>\phpMyAdmin</code>目录，不需要软件内管理数据库可略过此步。~~已集成adminer数据库管理软件，平替phpmyadmin。    2023/04/26</li>
<li>自行安装的集成环境，可能需要手动编辑<code>php.ini</code>解禁<code>exec</code>函数。以宝塔面板为例：进入PHP设置-&gt;禁用函数-&gt;删除<code>exec</code>。并且IP地址注册为主机名、尽量关闭跨站攻击防御。</li>
</ol>
<h4>Linux手动安装：</h4>
<ol>
<li>首先，执行《Windows手动安装》相同的步骤。</li>
<li>非Windows平台仅测试了Linux-x86_64平台，稍作调整即可直接使用：调整数据临时目录为相对路径，打开配置文件<code>\config\conf.php</code>修改<code>fileName0</code>变量即可，建议改为<code>$fileName0='../AccSoft_Data/';</code>，如果新建账套时出现错误，则尝试安装Mysql Community Client程序（Client版本尽量在5.7.17及以上）<a href="https://downloads.mysql.com/archives/community/">Mysql历史版本下载页</a><s>，安装无效可尝试将项目<code>\config</code>目录下的<code>mysql</code>和<code>mysqldump</code>文件拷贝到<code>/usr/bin/</code>并重启服务器。安装的Mysql Community Client程序必须适用于您的系统平台。</s></li>
<li>非Windows平台下，账套或其他文件的导出或有问题，但未做测试。有测试的朋友可帮忙将错误提示发送到issue页面反馈给我谢谢。</li>
</ol>
<ul>
<li>注意：本软件权限管理比较开放，增删账套时不需要后端初始化database密码，而是用户键入数据库密码登录，容易受中间人攻击技术劫持。所以用户角色权限管理和SSL证书高级加密尤为重要，且企业生产场景下不建议公开部署，以防会计信息泄露、丢失、注入；或者用户可以自行对以下路径文件进行二次开发，覆写敏感内容。</li>
</ul>
